## Introduction

[Algolia](https://www.algolia.com) is a search api that easily integrates into any application. It is a super powerful tool that allows you to scale your search content and provide a fast, easy to use search experience.

In this course we will add Algolia's InstantSearch to a Next.js application and cover all features. We will be using [this](https://github.com/vercel/next.js/tree/canary/examples/with-mdx-remote) Next.js `mdx` blog starter repo as a starting point.

## Level

This course is at the intermediate level. You should have a basic understanding of Next.js, JavaScript, and basic HTML/CSS.
