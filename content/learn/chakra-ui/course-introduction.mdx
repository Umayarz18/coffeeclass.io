## What's In This Course?

In this course you will learn Chakra UI. We'll start by installing Chakra UI in React, Next.js, and Gatsby. Next, we'll go over the basics and work our way through to more advanced concepts. Each module builds on the previous modules, however they can be completed independently or at random as well.

Here is some of what we'll cover:

- Chakra UI installation
- Major component overview
- Deep dive into `Flex` and `Grid` components
- Build a complete landing page using Chakra UI
- Chakra UI + SEO
- Add Dark Mode + A Dark Mode Switch
- And more...

## Course Features

- Free!
- Uses the latest versions of Chakra UI and Next.js

## Next.js

Through the course, we'll be using Next.js as the framework of choice. If you are not familiar with Next.js, you can read more about it [here](https://nextjs.org/docs/getting-started).
