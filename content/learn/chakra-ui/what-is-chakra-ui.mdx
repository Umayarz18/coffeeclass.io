[Chakra UI](https://chakra-ui.com/) is a CSS framework for React based applications (React, Next.js, and Gatsby) which allows you to create accessible React apps with speed.

> Chakra UI is a simple, modular and accessible component library that gives you the building blocks you need to build your React applications.
